* 저의 지식과 역량 부족으로 여러가지 추가하지 못한 기능들이 많고 
  프로그램 자체가 매우 미흡합니다. 선생님의 너그러운 마음으로 양해 부탁드립니다.

현재까지 추가한 것들

 - 한국어 <-> 영어 선택 박스 (모양만.)
 - C, D 드라이브 목록 표시 (최상위 폴더 목록만, 그 외 하위폴더는 표시하지 못함.) -> 수업시간에 만든 코드 사용
 - JFrame을 사용하여 창 닫기, 최소화, 최대화를 가능하게 함.

 - E, F 드라이브도 표시할 수 있게 함 (최상위 폴더 목록만, 그 외 하위폴더는 표시하지 못함.), D, E, F 드라이브가 없을 시 "찾을 수 없음" 출력하게 함.
 - 콤보박스에서 언어 변경시 타이틀명, 폴더명 등 선택한 언어로 변경할 수 있도록 구현하는중 (완료 x)
 - JPopmenu를 사용하여 마우스 우클릭시 창이 나타나도록 구현하는중 (완료 x)


만든이가 알고있는 버그, 오류들 등... (일부)

 - 언어 변경 부분을 제대로 구현하지 못하여 프로그램 실행시 오류가 발생할 수 있음 -> 관련된 코드를 제거하면 오류 없음
 - 드라이브의 하위폴더가 ABC순이 아닌 역순으로 표시됨 -> 수정 시도중
 - 하위 폴더, 파일 목록을 표시하지 못함 -> JTable을 사용하여 추가 예정
...
